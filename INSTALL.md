# Installing SeeCen (for sellers — no coding needed)

SeeCen runs on your own computer. Your store data stays on your machine and
is never uploaded anywhere.

## Easiest: install from the website

1. Open **https://seecen.seekerscentral.com** in Chrome or Edge.
2. Click **Install from browser** (or the install icon in the address bar).
3. SeeCen opens as its own app with an icon — it also works offline.

On iPhone/iPad: open the site in Safari → Share → **Add to Home Screen**.

## Windows app

1. Go to the [latest release](https://github.com/Draven047/SeeCen/releases/latest)
   and download the file ending in **`-setup.exe`**.
2. Double-click it. Windows may show **“Windows protected your PC”** because the
   app is new and unsigned — click **More info → Run anyway**.
3. Finish the installer. SeeCen appears in your Start menu.

## Mac app

1. From the [latest release](https://github.com/Draven047/SeeCen/releases/latest),
   download the **`.dmg`** file.
2. Open it and drag **SeeCen** into **Applications**.
3. The first time only: **right-click** the app → **Open** → **Open**
   (macOS asks this for new unsigned apps).

## Linux

Download the **`.AppImage`** from the latest release, make it executable
(`chmod +x`), and run it. A `.deb` is also provided for Debian/Ubuntu.

## First run

- Pick your language — English, हिन्दी, தமிழ், తెలుగు, বাংলা, or मराठी.
- Choose **Set up my store** and enter your store name — you start with a
  clean store. (Or explore with sample data first.)

## Keeping your data safe

- Everything lives on this computer only.
- Tap the **Demo** button (top-right) → **Export backup** to download a backup
  file. Keep it somewhere safe.
- Moving to a new computer? Install SeeCen there and use **Import backup**.

## Need help?

Tap the **?** button in the top bar for short “How do I…” guides, or open an
issue at https://github.com/Draven047/SeeCen/issues.
